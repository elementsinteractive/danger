## Installation

As a dev dependency

```bash
$ npm install -D @makerstreet/danger
```

If you use yarn:

```bash
$ yarn add @makerstreet/danger --dev
```

## CLI Usage

At a glance:

```js
// dangerfile.js
import dangerElements from '@elements/danger'

```
